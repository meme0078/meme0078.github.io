Quiz 3 - Machine Learning

Inicio
1. Tengo un modelo que marca los puntos faciales de una persona
A. Se hace con un modelo de detección de anomalías
B. Se basa en promedio móvil ponderado
C. Es un modelo de clasificación
D. Es un modelo de regresión
E. Todas las anteriores
F. Ninguna de las anteriores
Respuesta correcta: D
Marcar puntos faciales implica predecir coordenadas continuas en una imagen (como posición X e Y), lo cual es una tarea típica de regresión.
2. Es conocido como distancia Manhattan
A. L1 norm
B. L2 norm
C. RMSE
D. DBSCAN
E. Todas las anteriores
F. Ninguna de las anteriores
Respuesta correcta: A
La distancia Manhattan o "L1 norm" es la suma de los valores absolutos de las diferencias entre coordenadas.
3. Normalmente dividimos el dataset en 80 / 20. A este 20% se le conoce regularmente como
A. Training Set
B. Validation Set
C. Test Set
D. Cross-Validation Set
E. Todas las anteriores
F. Ninguna de las anteriores
Respuesta correcta: C
El 20% restante típicamente se reserva como conjunto de prueba (test set), para evaluar el desempeño final del modelo.
4. Un dato arroja -0.9 en su correlación contra otro dato. Esto significa
A. Relación directa alta
B. Relación directa baja
C. Relación inversa baja
D. Relación inversa alta
E. Todas las anteriores
F. Ninguna de las anteriores
Respuesta correcta: D
Un valor de -0.9 indica una fuerte correlación negativa. A medida que una variable aumenta, la otra disminuye con gran consistencia.
5. Cuando hacemos limpieza de datos tenemos información categórica que podemos ingresar directamente al modelo
A. Cierto
B. Falso
Respuesta correcta: B
Los algoritmos de machine learning no pueden manejar variables categóricas directamente. Estas deben ser transformadas mediante codificación.
6. Para entrenar al modelo utilizamos el siguiente método
A. .transform()
B. .fit()
C. .train()
D. .train_test()
E. Todas las anteriores
F. Ninguna de las anteriores
Respuesta correcta: B
En librerías como scikit-learn, `.fit()` es el método estándar para ajustar un modelo a los datos de entrenamiento.
7. Tengo un split del dataset y voy a aplicar validación cruzada, puedo aplicar esto haciendo
A. KFold
B. fit_transform
C. fit
D. transform
E. corr
F. Todas las anteriores
G. Ninguna de las anteriores
Respuesta correcta: A
`KFold` es una técnica de validación cruzada que divide los datos en K subconjuntos y entrena/evalúa el modelo múltiples veces.
8. Puedo obtener los mejores hiperparámetros del modelo AUTOMÁTICAMENTE haciendo
A. GridSearch
B. Manual Tunning
C. RandomizedGridSearch
D. Matriz de covarianza
E. Todas las anteriores
F. Ninguna de las anteriores
Respuesta correcta: A
GridSearch y RandomizedSearch son métodos automáticos para encontrar los mejores hiperparámetros. Manual tuning no es automático.
9. Para servir el algoritmo en una página web debemos obtener
A. El REST API
B. El modelo en un formato serializable
C. La aplicación de Android
D. La aplicación de React JS
E. Todas las anteriores
F. Ninguna de las anteriores
Respuesta correcta: A
El modelo debe exponerse mediante una API para ser consumido por una página web. También debe estar serializado para funcionar.
10. ¿Cuál de las siguientes no es una plataforma para servir una aplicación de IA?
A. GCP (Google Cloud Platform)
B. AWS (Amazon Web Services)
C. Microsoft Azure
D. Yahoo Web Services
E. Todas las anteriores
F. Ninguna de las anteriores
Respuesta correcta: D
Yahoo no ofrece actualmente una plataforma conocida para desplegar modelos de IA. Las otras sí son ampliamente utilizadas.
