Laboratorio 2 – Predicción de alquiler de bicicletas

Regresión con modelos Lineal, Árbol de Decisión y Random Forest

Inicio
¿Qué se hizo?

Se realizó una regresión utilizando datos del sistema de bicicletas compartidas, aplicando los siguientes pasos:

  • Análisis exploratorio de datos con histogramas y relaciones.
  • Transformaciones como la creación de la variable day_type.
  • Entrenamiento de modelos de regresión: Lineal, Árbol de decisión y Random Forest.
  • Evaluación con RMSE y validación cruzada.

¿Qué aprendí?

Este laboratorio permitió entender la aplicación de modelos de regresión, el uso de validación cruzada, y la importancia del análisis exploratorio para preparar los datos y elegir las variables más representativas.
