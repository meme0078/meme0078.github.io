Quiz 3 - Machine Learning

Inicio
1. ¿Qué es el machine learning?
A. Un conjunto de técnicas que permiten a las máquinas aprender a partir de datos
B. Un conjunto de técnicas que permiten a las máquinas procesar grandes cantidades de datos
C. Un conjunto de técnicas que permiten a las máquinas interpretar datos
Respuesta correcta: A
El machine learning permite que las máquinas aprendan patrones a partir de datos para realizar predicciones o tomar decisiones.
2. ¿Qué es un vector en matemáticas?
A. Una cantidad escalar
B. Una cantidad con dirección y magnitud
C. Una función matemática
Respuesta correcta: B
Un vector tiene magnitud y dirección, y se representa frecuentemente como una flecha en el espacio.
3. ¿Qué es una matriz en matemáticas?
A. Una cantidad escalar
B. Una cantidad con dirección y magnitud
C. Un arreglo n-dimensional de elementos numéricos
Respuesta correcta: C
Una matriz es una estructura de datos que organiza números en filas y columnas.
4. ¿Cuál es la operación matricial que representa la transposición de una matriz A?
A. A.T
B. A**-1
C. A**2
Respuesta correcta: A
La transposición se denota como A.T y cambia filas por columnas.
5. Si quisiéramos multiplicar 'A' y 2 ¿podríamos realizar la operación?
A. Sí
B. No
Respuesta correcta: A
Sí, si A es una matriz, puede multiplicarse por un escalar como 2.
6. ¿Qué es una función matemática?
A. Un conjunto de reglas que asignan a cada elemento de un conjunto de entrada un único elemento de un conjunto de salida
B. Un conjunto de números
C. Un conjunto de vectores
Respuesta correcta: A
Una función es una relación entre conjuntos donde cada entrada tiene una salida única.
7. ¿Qué es una función lineal en matemáticas?
A. Una función que tiene la forma f(x) = a*x + b
B. Una función que tiene la forma f(x) = ax**2 + b*x + c
C. Una función que tiene la forma f(x) = e**x
Respuesta correcta: A
La función lineal es una línea recta y su forma es f(x) = ax + b.
8. ¿Qué es una función exponencial en matemáticas?
A. f(x) = a*x + b
B. f(x) = ax**2 + b*x + c
C. f(x) = e**x
Respuesta correcta: C
La función exponencial se caracteriza por tener la variable como exponente.
9. ¿Qué es la regresión lineal en el contexto del machine learning?
A. Un algoritmo de clasificación
B. Un algoritmo que encuentra la mejor línea recta que describe la relación entre dos variables
C. Un algoritmo que usa una función exponencial
Respuesta correcta: B
La regresión lineal busca ajustar una línea recta que modele la relación entre variables.
10. ¿Cuáles son los tres tipos principales de machine learning?
A. Aprendizaje supervisado, no supervisado, reforzado
B. No supervisado, semisupervisado, supervisado
C. No supervisado, supervisado, profundo
Respuesta correcta: A
Los tres principales son supervisado, no supervisado y por refuerzo.
11. ¿Qué es el aprendizaje supervisado?
A. Aprendizaje con datos etiquetados
B. Aprendizaje con datos no etiquetados
C. Aprendizaje con datos aleatorios
Respuesta correcta: A
El aprendizaje supervisado usa ejemplos donde se conoce el resultado esperado.
12. ¿Qué es una función de pérdida?
A. Mide la discrepancia entre predicción y realidad
B. Mide la complejidad del modelo
C. Mide la velocidad de aprendizaje
Respuesta correcta: A
La función de pérdida ayuda a cuantificar cuán mal está funcionando un modelo.
13. ¿Cuál es el objetivo de la regresión logística?
A. Predecir valores continuos
B. Predecir valores discretos
C. Clasificar datos
Respuesta correcta: C
La regresión logística es usada para tareas de clasificación binaria o múltiple.
14. ¿Qué se puede resolver con la regresión logística?
A. Regresión lineal
B. Regresión no lineal
C. Clasificación binaria y múltiple
Respuesta correcta: C
La regresión logística se utiliza comúnmente en problemas de clasificación.
15. ¿Cuál es la función de pérdida utilizada en la regresión logística?
A. Entropía cuadrática
B. Entropía cruzada
C. Error cuadrático medio
Respuesta correcta: B
En clasificación, la entropía cruzada es ideal para medir la precisión de las probabilidades predichas.
16. ¿Cuál es el objetivo de la regresión lineal?
A. Predecir valores continuos
B. Predecir valores discretos
C. Clasificar datos
Respuesta correcta: A
La regresión lineal se utiliza para predecir valores numéricos continuos.
17. ¿Cuál es la función de costo utilizada en la regresión lineal?
A. Error cuadrático medio
B. Entropía cruzada binaria
C. Pérdida logarítmica
Respuesta correcta: A
El error cuadrático medio mide el promedio de los cuadrados de los errores en regresión.
18. ¿Cómo se puede evaluar si el modelo de regresión lineal se ajusta adecuadamente?
A. Inspección visual de residuos
B. Comparación de valores predichos y reales
C. Cálculo del error cuadrático medio
Respuesta correcta: C
El error cuadrático medio ofrece una medida cuantitativa del ajuste del modelo.
