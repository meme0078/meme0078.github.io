Resumen del Quiz - Machine Learning

Inicio
1. ¿Qué es Machine Learning?
A. Una configuración especial en las máquinas para habilitar su aprendizaje
B. Algoritmos que hacen que la máquina aprenda sin necesidad de hacer explícita la programación
C. Realizar programación por medio de IF ELSE para que la máquina aprenda
D. Realizar reglas de entrada y salida para que la máquina aprenda
E. Todas las anteriores
F. Ninguna de las anteriores
Respuesta correcta: B
El machine learning permite que un algoritmo aprenda de datos sin programación explícita con reglas.
2. Serie de algoritmos con datos y etiquetas como entrada, y modelo como salida
A. Supervised Learning
B. Unsupervised Learning
C. Semi supervised Learning
D. Reinforcement Learning
E. Todas las anteriores
F. Ninguna de las anteriores
Respuesta correcta: A
El aprendizaje supervisado usa datos con etiquetas conocidas para entrenar modelos predictivos.
3. Cuando ingresamos datos categóricos y la salida es un número
A. Clasificación
B. Regresión lineal
C. Regresión Logística
D. Bagging
E. Todas las anteriores
F. Ninguna de las anteriores
Respuesta correcta: B
La regresión lineal predice valores continuos incluso a partir de datos categóricos codificados.
4. Algoritmos con recompensa y pérdida mediante políticas
A. Machine Learning
B. Supervised Learning
C. Reinforcement Learning
D. Clasificación
E. Todas las anteriores
F. Ninguna de las anteriores
Respuesta correcta: C
El aprendizaje por refuerzo aprende acciones óptimas con base en recompensas y penalizaciones.
5. En batch learning aprendemos en el vuelo
A. Cierto
B. Falso
Respuesta correcta: B
Batch learning entrena con el dataset completo, no en tiempo real como en online learning.
6. Uso de vector de características para reidentificación
A. Aprendizaje basado en modelo
B. Aprendizaje basado en instancia
C. Aprendizaje automático
D. Aprendizaje estocástico
E. Todas las anteriores
F. Ninguna de las anteriores
Respuesta correcta: B
En el aprendizaje basado en instancia, se compara con ejemplos previos usando vectores.
7. ¿Cuál NO es un algoritmo no supervisado?
A. PCA
B. Eclat
C. t-SNE
D. SVM de 1 clase
E. Regresión logística
F. Todas las anteriores
G. Ninguna de las anteriores
Respuesta correcta: E
La regresión logística es un algoritmo supervisado de clasificación binaria.
8. Accuracy alto en entrenamiento y bajo en test set
A. Underfitting
B. High bias
C. High variance
D. Overfitting
E. Todas las anteriores
F. Ninguna de las anteriores
Respuesta correcta: D
El modelo se sobreajustó a los datos de entrenamiento y no generaliza bien a nuevos datos.
9. Baja precisión en validation y test set
A. Underfitting
B. Overfitting
C. High bias
D. High variance
E. Todas las anteriores
F. Ninguna de las anteriores
Respuesta correcta: A
El modelo es demasiado simple o mal entrenado, y no logra capturar los patrones.
10. Entrené con 100% de los datos y luego saqué 20% como test set
A. El modelo está bien hecho, hemos validado el training set con el test set
B. El modelo bien hecho, ya podemos ponerlo en producción
C. El modelo está mal hecho, lo hemos tenido que dividir en 98%/1%
D. El modelo está mal hecho, hemos de utilizar parte del dataset para training y la otra parte no vista por el modelo como test set
E. Todas las anteriores
F. Ninguna de las anteriores
Respuesta correcta: D
El test set debe estar completamente separado y no usado nunca en entrenamiento, para evaluar el rendimiento real.
